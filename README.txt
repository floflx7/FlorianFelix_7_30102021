Groupomania
Réseau social d'entreprise

Fonctionnalités
L'utilisateur doit s'inscrire ou s'authentifier pour accéder au site
L'utilisateur authentifier peut ajouter des posts, commenter et liker/disliker des posts
Seul l'utilisateur  ayant créer le post et l'administrateur du site peuvent modifier ou supprimer le post
L'utilisateur peut voter à sa guise pour les sauces de son choix


Lancer localement le frontend
Cloner le projet dans un repertoire

git clone https://github.com/floflx7/FlorianFelix_7_30102021.git

Aller dans le répertoire frontend

cd frontend
installer les dépendances

npm install

  
npm start

  
Lancer localement le backend
Aller dans le répertoire backend

  cd backend
Installer les dépendences

  npm install
Démarrer le serveur

  nodemon server



Server: NodeJS, ExpressJS, MySql
